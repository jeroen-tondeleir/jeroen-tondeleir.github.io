IT-profile

May 10, 2016

Hier kunt u een beeld vormen van mijn vaardigheid in een bepaalde software of een programma. Deze cijfers worden weergeven op een schaal van 0 tot 100%

Legende

0% ik kan/begrijp er niks van.
20%: ik begrijp het, maar ik kan het moeilijk gebruiken.
40%: ik kan het gebruiken a.d.h.v. internet, boeken en andere documentatie.
60%: ik kan het gebruiken op een beginnersniveau.
80%: ik kan het tamelijk goed gebruiken.
100%: ik kan het zéér goed gebruiken.
