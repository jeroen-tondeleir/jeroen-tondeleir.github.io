IT-profile

Hier kunt u een beeld vormen van mijn vaardigheid in een bepaalde programmeertaal. Deze cijfers worden weergeven op een schaal van 0 tot 100%

Legende

0% ik kan/begrijp er niks van.
20%: ik begrijp de taal, maar ik kan hem moeilijk gebruiken.
40%: deze taal kan ik gebruiken a.d.h.v. internet, boeken en andere documentatie.
60%: deze taal kan ik gebruiken op een beginnersniveau.
80%: ik kan deze taal tamelijk goed.
100%: ik kan deze taal zéér goed.
